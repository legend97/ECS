Blocks

The block element is not an actual element that changes the page. The block element is an element that is most commonly used for "enhancements," because of its ability to classify different parts into "blocks," hence its name.

<div style="color:blue;">Daniel</div>
<div"color:green;">Li</div>

The code above will change "Daniel" with blue font, and "Li" with green font.
